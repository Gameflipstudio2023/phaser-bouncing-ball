Welcome to Mincrosoft Explorer!

This is a 90s-style web browser simulation. While it can't actually browse the web, it demonstrates what browsers looked like in the 1990s.

Features:

  • Retro 90s design
  • Tab system (click + to add new tabs)
  • Address bar
  • Navigation buttons
  • Window controls
